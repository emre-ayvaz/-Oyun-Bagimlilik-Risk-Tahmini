Web uygulamasını çalıştırmak için Komut İstemi açılıp önce proje klasörüne gidilmelidir.

Örnek:

cd /d "C:\Users\EMRE\Desktop\proje_klasoru"

Bu yol örnektir. Proje klasörü hangi konumdaysa o klasörün yolu yazılmalıdır.

Daha sonra uygulama aşağıdaki komutla çalıştırılır:

streamlit run app.py

Komut çalıştıktan sonra uygulama tarayıcıda otomatik açılır. Açılmazsa şu adresten açılabilir:

http://localhost:8501

Not: app.py, model dosyası ve gerekli dosyalar aynı klasör içinde olmalıdır.
